# AGM 信念修正分类：完整决策树

Phase 3 执行时阅读此文件。目标是把不同时期的观点差异分成可复用的认知变化类型，而不是把所有差异都解释成"他变了"。

## 理论背景

AGM 理论（Alchourrón, Gärdenfors, Makinson, 1985）用于描述信念变化。数字炼化采用四类操作：

- **扩展（Expansion）**：加入新信念，原有信念不受影响。
- **修正（Revision）**：加入新信念，同时调整已有信念以维持一致性。
- **收缩（Contraction）**：放弃某个信念。
- **分裂（Split）**：认知层面维持旧信念，行为层面违反旧信念。

## 决策树

### 入口条件

检测到时间点 A 的观点 X 与时间点 B 的观点 Y 存在矛盾，且 B 晚于 A。

```text
Step 1: X 和 Y 是否处于相同或相似语境？
        （相似的市场环境、受众、问题类型）

├─ YES: 相同或相似语境
│   │
│   ├─ 对象在 B 时的言论是否仍支持 X，但行为违反 X？
│   │   ├─ YES → 【分裂 Split】
│   │   │   说明：认知与行为不一致，且未自觉反思
│   │   │   处理：不用于阶段划分，记入认知偏误画像
│   │   │
│   │   └─ NO → 继续判断
│   │
│   ├─ Y 是 X 的超集？
│   │   （Y 包含 X 的核心内容，并增加新维度、新条件或新推论）
│   │   ├─ YES → 【扩展 Expansion】
│   │   │   说明：在原有认知上叠加新层次，没有否定旧认知
│   │   │
│   │   └─ NO → Y 和 X 在相同论题上给出不同结论
│   │       → 【修正 Revision】
│   │       说明：新认知替代旧认知在同一问题上的判断
│   │
│   └─ 补充检查：Y 是否完全否定 X 的某个前提？
│       ├─ YES → 【修正 Revision】，且可能是重大修正
│       └─ NO → 回到上面的判断
│
└─ NO: 不同语境
    │
    ├─ 矛盾能否用语境差异完全解释？
    │   │
    │   ├─ YES → 标注"语境依赖性差异"
    │   │   说明：不是真正的矛盾，而是同一框架在不同环境下的应用
    │   │   处理：不纳入 AGM 分类，在语境标注中记录
    │   │
    │   └─ NO → 语境差异不足以解释矛盾
    │       → 继续 Step 2
    │
    └─ Step 2: 旧观点 X 是如何消失的？
        │
        ├─ 对象言论仍支持 X，但行为违反 X？
        │   └─ YES → 【分裂 Split】
        │
        ├─ 有明确替代
        │   （对象自述"我以前错了"或出现明确的新行为模式）
        │   → 【修正 Revision】
        │
        └─ 静默消失
            （观点 X 在某时间点后不再出现，且没有明确替代）
            → 【收缩 Contraction】
```

## 置信度三档

每次 AGM 分类必须附带置信度标注：

| 置信度 | 标准 | 在阶段划分中的权重 |
|--------|------|-------------------|
| **High** | 有明确自述、可观测行为改变、时间点清晰 | 可作为阶段分界依据 |
| **Medium** | 有两项证据，但缺第三项 | 可作为阶段内重要事件，不单独作为分界依据 |
| **Low** | 仅有推理支撑，缺乏直接证据 | 只作参考，不用于阶段划分或心智模型提取 |

只有 High 和 Medium 的 AGM 分类才能用于 Phase 3 阶段划分。

## 失效时间维度

每条 AGM 分类都记录 `obsolescence_window` 字段，用于描述旧观点到新观点之间的时间跨度。

### 沉默期处理规则

如果 T1（旧观点时间）和 T2（新观点时间）之间存在超过 3 年的沉默期，且 T2 的新观点没有明确的"我以前错了"自述：

→ 默认分类为收缩（Contraction）而非修正（Revision）。

理由：长期沉默后的立场变化，在缺乏自述反思时，更可能是"不再坚持"而非"有意识地替代"。

## 分裂类型

### 判定标准

分裂（Split）必须同时满足：

1. 对象在 T2 时的言论仍然支持 T1 的观点。
2. 对象在 T2 时的行为证据显示实际操作与言论相反。
3. 对象未明确反思这种不一致。

### 与其他类型的区分

| 特征 | 修正 | 收缩 | 分裂 |
|------|------|------|------|
| 言论变化 | 是 | 消失 | 否，仍支持旧观点 |
| 行为变化 | 是 | 不确定 | 是，违反旧观点 |
| 自觉反思 | 通常有 | 通常无 | 无 |

### 在炼化中的处理

- 不用于阶段划分。
- 记入认知偏误画像。
- 可用于价值张力提取。

## 认知跃迁 vs 观点变化

一系列 AGM 修正可能指向更深层的认知跃迁。但认定跃迁的门槛高于认定修正。

| 维度 | 观点变化（AGM 修正） | 认知跃迁（Kegan 质变） |
|------|---------------------|------------------------|
| 变化范围 | 局部，某个具体问题 | 系统性，影响多个领域 |
| 自觉性 | 可能不自觉 | 通常伴随元认知反思 |
| 证据要求 | 前后观点不同即可 | 至少 3 个条件同时满足 |
| 处理方式 | 同一阶段内的修正 | 阶段分界线 |

### 认知跃迁的必要条件

1. **自述框架级反思**：对象明确表达过"我以前的整个思考方式有问题"或等效表述。
2. **多领域行为同时变化**：不只是一个决策变了，而是多个领域的决策模式同时转变。
3. **变化不可逆**：此后再未回到旧框架。

三个条件缺任何一个，降级为观点变化。

## 特殊情况处理

### 情况 1：渐进式变化

有时候观点变化不是一个明确时间点，而是渐进过程。处理方式：

- 标注变化起点：最早出现新倾向的证据。
- 标注变化终点：旧观点最后出现的证据。
- 在这个时间段内标注为"过渡期"。

### 情况 2：表面矛盾实为多层次

看似矛盾的言论可能处在不同抽象层次：

- 战略层："长期持有好公司"。
- 战术层："现在应该增加现金比例"。

这不是 AGM 修正，而是不同层次的一致性应用。

### 情况 3：矛盾来自来源可信度差异

如果矛盾的两个观点来自不同可信度来源，优先信任可信度高的来源，将低可信度来源标注为"待验证"。

### 情况 4：言行不一致程度判断

并非所有言行不一致都构成分裂：

- 行为偏离较小或可由语境差异解释：不标注为分裂。
- 行为偏离显著且同语境下言行不一致：标注为分裂。
- 对象后来承认这种不一致并形成新认知：可改判为修正。
